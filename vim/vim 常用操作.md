# VIM 常用操作

## 位置符号

```
- 0                 将光标移动到本行的行首
- $                 将光标移动到本行的行尾
- H                 将光标移动到屏幕开头的行首
- M                 将光标移动到屏幕中间的行首
- L                 将光标移动到屏幕末尾的行首
- zz                将光标所在行`调整`到屏幕中央
```

## 单词间移动

```
- w/W               移动到下一个单词的开头
  > W               以空白字符为分割
- b/B               移动到上一个单词的开头
- e/E               移动到当前单词结尾
```

## 段落间移动

```
- ( | )             句子间移动
- { | }             段落间移动
```

## 行号跳转

```
- {行号}G            跳转到第{行号}行
- {下面几行}<回车>    光标跳转到{下面几行}
```

## 查找

```
- 正向搜索           /{要查找的内容} n 向下找 N 向上找
- 逆向搜素           ?{要查找的内容} n 向上找 N 向下找
- 快速逆向搜索        将光标移动到想查找的单词上 按下 #
```
## 插入

```
按下 i 键，在光标前插入。
按下 a 键，在光标后插入。
按下 I 键，在行首插入。
按下 A 键，在行尾插入。
按下 o 键，在光标下方打开新的一行 并进入插入模式。
按下 O 键，在光标上方打开新的一行 并进入插入模式
```
## 替换
> 默认 正则表达式需要转义

> \v 开启不用转义
```
- r{替换的一个字母或一个字}                                         替换一个字符
- R                                                             进入替换模式 光标处的内容直接替换 覆盖当前字符
- :s/{正则表达式(old)}/{替换的内容(new)}                          替换光标所在行的第一个匹配的内容
- :s/{正则表达式(old)}/{替换的内容(new)}/g                        替换光标所在行的所有匹配的内容
- :s/{正则表达式(old)}/{替换的内容(new)}/gc                       替换光标所在行的所有匹配的内容 并在替换前询问
- :{开始行号},{结束行号}s/{正则表达式(old)}/{替换的内容(new)}/g      替换两行之间出现的每个匹配串
- :%s/{正则表达式(old)}/{替换的内容(new)}/g                       替换整个文件的所有匹配项
```
## 删除
```
- x                 删除一个字符
- {数量}x            删除{数量}个字符
- dd                删除光标所在行
- {数量}dd           删除光标所在行在内的{数量}行
- dw                删除光标到下一个单词开头之间的内容
- de                删除光标到一个单词末尾之间的内容
- db                删除光标到上一个单词末尾的内容
- D                 删除光标到该行末尾的内容
- {数量}D            删除光标到包括其在内的{数量}行末尾的内容

```
## 粘贴
```
- p                 当复制的 不是 行时，内容插入到光标后。
                    当复制的  是  行时，内容出入到光标的下一行。
- P                 当复制的 不是 行时，内容插入到光标前。
                    当复制的  是  行时，内容出入到光标的上一行。       
```
